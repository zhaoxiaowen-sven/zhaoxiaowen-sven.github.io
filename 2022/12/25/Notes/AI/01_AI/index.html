一、机器学习的分类

1.1、分类

Regression：The function outputs a scalar.

Classification：Given options (classes), the function outputs the correct one.

Structured Learning：create something with structure (image, document)

1.2、机器学习步骤

1、Function With UnKownn Parameters

Model : y = b + wx

y : 预测结果, x : 输入参数, w : weight, b：bias

2、Define Loss from Training Data

Loss：how good a set of values is。L(w，b)

误差：e1 = |y - y’| , L means MAE

​ e2 = (y - y’)^2 , L means MSE

Loss =
$$
L = 1/N \sum_1^n e_n
$$

3、Optimization

Gradient Descent

梯度下降，找出Loss 最小的 w 和 b
