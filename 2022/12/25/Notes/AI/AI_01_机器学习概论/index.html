AI_01_机器学习概论

一、机器学习的分类

1.1、分类

Regression：The function outputs a scalar。找函数关系，输出一个数字

Classification：Given options (classes), the function outputs the correct one。做选择题， 从选项中选择

Structured Learning：create something with structure (image, document)。创造

1.2、机器学习步骤

1、Function With UnKownn Parameters

Model（函数） : y = b + wx

y : 预测结果, x : 输入参数, w : weight, b：bias

2、Define Loss from Training Data

Loss（误差）：how good a set of values is。

计算误差的公式：Loss = L(w，b)
$$
L = 1/N \sum_1^n e_n
$$

e1 = |y - y’| , L means MAE，绝对值差

e2 = (y - y’)^2 , L means MSE，绝对值差的平方

3、Optimization

Gradient Descent：梯度下降，找出误差最小的 Model

随机选取一个w0，计算w对Loss的微分（斜率）n，n > 0, 增大 W，n < 0，减小W。

1.3、机器学习步骤理解

1、new Function

简单函数模拟不出来红色的复杂的情况，这叫做model的 bias，需要使用S型的简单函数进行叠加。

red = constant + sum of set of f（x）

Sigmoid Function

S型的函数
$$
y = c * 1/1+e^{-(b + wx_1)} = c * sigmoid(b + wx_1)
$$

w决定函数的斜率，b决定函数的x起始点，c 决定函数的Y的最大高度

red = constant + setof sigmoid function ，红色的函数是一系列S型函数叠加的结果。

$$ y = b + \sum_i c_i sigmoid(b_i + w_ix_1) $$ 使用S型函数可以写出弹性非常好的复杂函数。

2、Loss

$$
Loss = L(\theta),\theta 代表所有未知的参数，
$$

3、Optimization

$$
\theta^* = arg min_\theta L , 找一组 \theta，使得Loss 最小
$$

步骤1 :

随机选一组初始值，计算 g

$$
g（Gradient， 梯度） = \nabla L(\theta^0),
\theta^0 代表的是一组参数，即每一组 \theta ^ {1..n} 对 L 进行微分，集合起来就是一个向量
$$

步骤2：

update 参数，采用刚才的方法将所有的 未知参数 集合起来，直到 g 是Zero Vector, 或者没法更低。
$$
\theta^0\ -> \eta -> \theta^1 ， \eta 表示机器学习的learningRate，过程表示一组数据集，计算 Loss 最小值的过程。
$$
